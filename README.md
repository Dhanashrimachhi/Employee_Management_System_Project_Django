## Installation 📦

>Download and Extract the provided source code zip file.

#### Clone

- Clone this repo to your local machine.

#### Run server locally

```shell
$ pip install Django
$ python manage.py migrate
$ python manage.py runserver
```

> Go to http://localhost:8000/ or http://127.0.0.1:8000/
